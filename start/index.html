Let’s
Get
Started

Bowery is a limitless development environment. Run your application in seconds no matter the operating system, programming language, database, or other dependencies. We're dedicated to making every stack easy to use.

Before we get going, you'll need:

  1. Internet
  2. AWS Account
  3. AWS Security Credentials

Setup

To start using Bowery:

  1. Download the app by clicking on the button below
  2. Unzip it and move it into your prefered directory (ie /Applications)
  3. Sign in with your existing Bowery credentials or signup

Connect AWS

You’ll be prompted for your AWS credentials. You will need an AWS account to use Bowery, so you can sign up here.

Bowery will run your applications on instances in your AWS account, so you’re the only one who has access to them. Keep in mind, AWS will charge 3 cents / hour for each Bowery app.

Add An App

The first step in creating an app is choose an image. You can search for various images both on the images pageand in the app.

Once you click on an image it will provision your app using that image. This may take a minute depending on how busy AWS is.

Update Info

Bowery will restart your app on file changes, so we need to know what command to run. Where your code should be placed on the AWS instance, and where the code is on your computer.

Once you’ve updated this information you’ll be able to edit your code and instantly see the changes.

Execute Commands

You can run commands on the AWS instance directly from Bowery. At the bottom of the edit app page click on the 'Run Commands' button.

Next Steps

You can see the app at the IP address under the app name in the apps view. It's public so you can share it with anyone.

If you give your image a name then other people will be able to create apps identical to your environment, including all changes you've made through the exec screen.

and you’re setup!
