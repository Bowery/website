Bowery
Terminal

By keeping code, software and operating systems are all in-sync across your team and organization, Bowery helps you:

  • Easily onboard new engineers
  • Improve software quality and delivery

Getting Started

Download the Bowery app by clicking on the button below.

Unzip and run the installer.

If you are on a Mac, it will save to your Applications folder. For Linux and Windows users it will save to your desktop.

Add An App

When the file dialogue opens, pick your app directory. This is where you will save your app. Wait a couple seconds (loading screen will show).

Your new terminal environment will pop up on your screen.

Start Coding

Now you can start coding. A .bowery file will be added in your directory.

You can use Github or any code-collaboration system like Bitbucket you use to distribute with your team.

When another person on your team selects the directory, all the installation and setup will be instantly available to them through the Bowery app.

and you’re setup!
